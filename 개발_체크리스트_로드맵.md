# 빗썸 원화마켓 자동매매 시스템 개발 체크리스트

> 📅 **전체 개발 기간**: 약 45일 (9주)
> 🎯 **목표**: 빗썸 KRW 마켓 전용 24/7 자동매매 시스템 구축
> ⚡ **핵심 전략**: EMA 크로스오버 + RSI 필터 + ATR 리스크 관리

---

## 📁 프로젝트 구조
```
bithumb-auto-trading/
├── src/
│   ├── core/           # 핵심 비즈니스 로직
│   ├── exchange/       # 거래소 연동
│   ├── data/           # 데이터 처리
│   ├── backtest/       # 백테스트
│   ├── ui/             # 사용자 인터페이스
│   └── jobs/           # 스케줄 작업
├── configs/            # 설정 파일
├── tests/              # 테스트 코드
├── scripts/            # 유틸리티 스크립트
└── docs/               # 문서
```
